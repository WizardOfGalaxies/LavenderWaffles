FONTLOG for LavenderWaffles

This file provides detailed information on the LavenderWaffles.
This information should be distributed along with LavenderWaffles and any derivative works.

Basic Font Information

LavenderWaffles supports the following characters: abcdefghijklmnopqurstuvwxyz0123456789.,!?+-_()[]<>/\@*#':;
all uppercase characters will display as the lowercase version, for simplicity
